Tutorial

A complete walkthrough using the synthetic test dataset included with sRNA-TAPS. By the end you will have run the full pipeline from raw FASTQs to called m5C sites and verified that the TAPS signal is working correctly.

Before you start: You need a Bowtie1 index and Ensembl GRCh38 GTF. The test FASTQs use real hg38 coordinates and will align to a standard hg38 index.
1

Generate the test FASTQs

The simulator generates 9 samples — 3 conditions × 3 replicates — with accurate TAPS chemistry. Known m5C positions are seeded into the templates so you can verify the signal end-to-end.

python3 tests/simulate_taps_srna.py \
    --outdir ~/srnataps_tutorial/fastq \
    --reads  1000000 \
    --seed   42

You should see 9 .fastq.gz files and a samples.tsv written to the output directory.

Tip: The --seed 42 flag makes the simulation reproducible. You will get identical results every time.
2

Adapter trimming

TrimGalore removes the TruSeq small RNA 3′ adapter and quality-trims the reads. The --small_rna flag sets the correct defaults for this library type.

mkdir -p ~/srnataps_tutorial/03.trimGalore

for fq in ~/srnataps_tutorial/fastq/*.fastq.gz; do
    trim_galore --small_rna \
        --length 18 --max_length 50 \
        --output_dir ~/srnataps_tutorial/03.trimGalore \
        $fq
done

Each sample will produce a _trimmed.fq.gz file and a trimming report. Expect ~80–90% of reads to pass the length filter.

3

Bowtie1 alignment

Align to the hg38 genome. The --norc flag is critical — small RNA libraries are strand-specific and reverse complement alignment would introduce false positives.

BOWTIE_IDX=/path/to/hg38_bowtie1_index
mkdir -p ~/srnataps_tutorial/04b.aligned

for fq in ~/srnataps_tutorial/03.trimGalore/*_trimmed.fq.gz; do
    sample=$(basename $fq _trimmed.fq.gz)
    bowtie -x $BOWTIE_IDX -q $fq \
        --norc -v 2 -k 10 --best --strata -m 100 \
        -S ~/srnataps_tutorial/04b.aligned/${sample}.sam \
        2> ~/srnataps_tutorial/04b.aligned/${sample}.log
    samtools sort -o ~/srnataps_tutorial/04b.aligned/${sample}.sorted.bam \
        ~/srnataps_tutorial/04b.aligned/${sample}.sam
    samtools index ~/srnataps_tutorial/04b.aligned/${sample}.sorted.bam
    rm ~/srnataps_tutorial/04b.aligned/${sample}.sam
done

Check the alignment rate for one sample:

cat ~/srnataps_tutorial/04b.aligned/treat_HEK_R1.log
Expected: ~80% alignment rate with the synthetic data.
4

Biotype annotation

Each read is assigned to the highest-priority overlapping biotype using the Ensembl GTF. Separate BAMs are written per biotype.

GTF=/path/to/Homo_sapiens.GRCh38.112.gtf
mkdir -p ~/srnataps_tutorial/05.biotype_bams

for bam in ~/srnataps_tutorial/04b.aligned/*.sorted.bam; do
    sample=$(basename $bam .sorted.bam)
    python3 05_annotate_biotype.py \
        --bam $bam --gtf $GTF \
        --out_dir ~/srnataps_tutorial/05.biotype_bams \
        --sample $sample
done

# Summarise biotype composition
python3 05_biotype_summarise.py \
    --in-dir ~/srnataps_tutorial/05.biotype_bams
5

Build the SNP blacklist

Merge the three no-treat replicates and call cell-line-specific C→T variants. These positions will be excluded from TAPS calling.

mkdir -p ~/srnataps_tutorial/06.snp_resources

samtools merge ~/srnataps_tutorial/06.snp_resources/notreated_HEK_merged.bam \
    ~/srnataps_tutorial/04b.aligned/no-treat_Ctrl_HEK_R1.sorted.bam \
    ~/srnataps_tutorial/04b.aligned/no-treat_Ctrl_HEK_R2.sorted.bam \
    ~/srnataps_tutorial/04b.aligned/no-treat_Ctrl_HEK_R3.sorted.bam
samtools index ~/srnataps_tutorial/06.snp_resources/notreated_HEK_merged.bam

python3 06_build_snp_blacklist.py \
    --bam  ~/srnataps_tutorial/06.snp_resources/notreated_HEK_merged.bam \
    --fasta /path/to/hg38.fa \
    --out  ~/srnataps_tutorial/06.snp_resources/sample_snps_HEK.bed \
    --min-af 0.20 --min-cov 5 --cell-line HEK
6

TAPS modification calling

Run the caller on each biotype BAM. SNP-flagged positions are excluded before any counting.

mkdir -p ~/srnataps_tutorial/07.taps_calls/miRNA

for bam in ~/srnataps_tutorial/05.biotype_bams/miRNA/*.sorted.bam; do
    sample=$(basename $bam _miRNA.sorted.bam)
    python3 07_taps_calling.py \
        --bam $bam \
        --fasta /path/to/hg38.fa \
        --out ~/srnataps_tutorial/07.taps_calls/miRNA/${sample}_miRNA_taps.tsv \
        --min-cov 3 --context ALL \
        --sample-snp-bed ~/srnataps_tutorial/06.snp_resources/sample_snps_HEK.bed \
        --cell-line HEK
done
7

Verify the TAPS signal

Check the two seeded m5C positions — chr17:59841313 in hsa-miR-21-5p and chrX:66018955 in hsa-miR-223-3p:

grep "^17.*59841313" \
    ~/srnataps_tutorial/07.taps_calls/miRNA/treat_HEK_R1_miRNA_taps.tsv

grep "^17.*59841313" \
    ~/srnataps_tutorial/07.taps_calls/miRNA/no-treat_Ctrl_HEK_R1_miRNA_taps.tsv

7b

Run benchmarking (optional)

sRNA-TAPS includes a benchmarking suite comparing calls against rastair, asTair, and Bismark. rastair is run twice — once in CpG-only mode and once in all-context mode — to demonstrate that rastair is a CpG-only caller by design.

# Run rastair (CpG-only and all-context)
sbatch 08a_rastair.sl

# Run asTair (all contexts)
sbatch 08b_astair.sl

# Run Bismark (bisulfite contrast, small RNA Bowtie2 flags)
sbatch 08c_bismark.sl

# Run comparison
python3 09_compare.py

The benchmarking produces four output directories:

08.benchmark/rastair/cpg/   — rastair CpG-only calls
08.benchmark/rastair/all/   — rastair all-context calls (identical to cpg)
08.benchmark/astair/        — asTair all-context calls per biotype
08.benchmark/bismark/       — Bismark bisulfite calls (chemistry inverted)
Key finding: rastair/cpg/ and rastair/all/ produce identical outputs, confirming rastair is a CpG-only caller regardless of flags. This demonstrates why a dedicated small RNA pipeline is needed for non-CpG m5C detection. See the Benchmarking page for full details.
8

Generate the report

Run the R report scripts to generate publication-ready figures.

RDIR=/path/to/sRNA-TAPS/srnataps/report/R
export SRNATAPS_R_DIR=$RDIR

Rscript $RDIR/run_all.R \
    --outdir  ~/srnataps_tutorial \
    --figdir  ~/srnataps_tutorial/report/figures \
    --scripts $RDIR \
    --skip-bench --skip-logos

Figures will be written to ~/srnataps_tutorial/report/figures/ in PDF, PNG, and SVG format.
