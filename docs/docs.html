Documentation

Complete reference for the sRNA-TAPS pipeline — installation, configuration, pipeline steps, and output formats.

Installation

Conda (recommended)

Installs sRNA-TAPS and all bioinformatics dependencies automatically.

conda install -c bioconda -c conda-forge srna-taps

pip

pip install sRNA-TAPS

From source

git clone https://github.com/HenzelerB/sRNA-TAPS
cd sRNA-TAPS
pip install -e .

R packages

Required for report figure generation (PDF/PNG/SVG).

Rscript install_R_packages.R

Quick Start

# 1. Initialise a new project
srnataps init \
    --outdir  ~/my_project \
    --genome  /path/to/hg38.fa \
    --gtf     /path/to/hg38.gtf

# 2. Edit the sample sheet
nano ~/my_project/samples.tsv

# 3. Run on SLURM
srnataps run --configfile ~/my_project/config.yaml --slurm

# 4. Run with benchmarking
srnataps run --configfile ~/my_project/config.yaml --slurm --benchmark

Pipeline Overview

rawfiles/      Raw FASTQs (SE, TruSeq small RNA)
    ↓
01. FastQC     Pre-trim QC
02. Trim       TruSeq SR adapter trimming
03. Align      Bowtie1: -v2 --norc -k10 --best --strata -m100
04. Biotype    miRNA > tRNA > piRNA > snoRNA > snRNA > rRNA > lncRNA > other
05. SNP        3-layer: dbSNP + cell-line + heterozygosity
06. Call       Binomial test + BH FDR correction
    ↓
07. Benchmark* rastair · asTair · Bismark
08. Compare*   Concordance · correlation
09. Report     R figures + HTML

*requires --benchmark flag

Step 3 — TAPS-aware Alignment

Four parameters are set specifically for TAPS data:

ParameterPurpose
-v 2Up to 2 total mismatches — tolerates C→T TAPS transitions
--norcForward strand only — prevents false G→A calls on reverse strand
-k 10 --best --strataUp to 10 alignments, writes XA:i:N tag for fractional weighting
--samRequired for XA tag output

Step 4 — Biotype Annotation

Reads are assigned to the highest-priority overlapping biotype:

miRNA > tRNA > piRNA > snoRNA > snRNA > rRNA > lncRNA > other
Note: TAPS enriches for rRNA relative to untreated libraries. Biotype proportions will differ between conditions — this is expected and not a sign of a failed experiment.

Step 5 — SNP Filtering

LayerMethodFlag
1dbSNP common C→T/G→A variants (AF ≥ 1%)SNP_KNOWN
2Cell-line SNPs from no-treat BAMsSNP_SAMPLE
3No-treat C→T rate ≥ 40%SNP_HET

The Delta (δ) Score

δ = treat_mod_rate − pb_Ctrl_mod_rate

Pyridine borane causes ~8–9% non-specific C→T conversion at unmodified cytosines. Subtracting pb_Ctrl removes this background, leaving only TET-dependent 5mC/5hmC signal.

δ valueInterpretation
~0.00No methylation above background
0.10–0.20Low-level methylation
0.20–0.40Moderate methylation — confident candidate
0.40–0.70High methylation
> 0.70Near-stoichiometric methylation

Test Dataset

python3 tests/simulate_taps_srna.py \
    --outdir /path/to/test_fastq \
    --reads  100000 \
    --seed   42

Generates 9 samples (3 conditions × 3 replicates) with accurate TAPS chemistry. Biotypes: miRNA 40%, tRNA 25%, rRNA 25%, snoRNA 10%.
