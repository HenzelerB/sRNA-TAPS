Chemistry

How TAPS works

🧫

Step 1 — TET oxidation

TET enzymes oxidise 5mC and 5hmC to 5-carboxylcytosine (5caC), marking modified bases for conversion.

⚗️

Step 2 — Pyridine borane reduction

Pyridine borane reduces 5caC to dihydrouracil (DHU), which is read as T during PCR amplification.

🔬

Result — C→T at modified sites

Unmodified cytosines remain as C. Only 5mC and 5hmC appear as T in the sequencing reads.

Inverse of bisulfite sequencing. In bisulfite, unmodified C converts to T and 5mC is protected. In TAPS, modified C converts to T and unmodified C is protected — making short RNA reads far easier to align.

Features

Built for small RNA

Every design decision in the pipeline accounts for the specific challenges of short, strand-specific, multimapping RNA molecules.

🎯

7 RNA biotypes

miRNA, tRNA, rRNA, snoRNA, snRNA, piRNA, lncRNA — each processed separately with biotype-appropriate coverage thresholds.

🛡️

3-layer SNP filtering

dbSNP common variants, cell-line-specific SNPs from no-treat BAMs, and heterozygosity detection — all applied before any C→T counting.

⚖️

Multi-mapper weighting

Reads with XA:i:N tags contribute 1/N to each locus, preventing inflation at tRNA gene copies and rRNA repeats.

📊

75 publication figures

R-generated PDF, PNG, and SVG figures at 300 dpi — QC, biotype composition, modification rates, benchmarking, and sequence logos.

🔄

Benchmarking suite

Built-in comparison against rastair, asTair, and Bismark with concordance heatmaps and Pearson correlation at shared sites.

🧬

Synthetic test dataset

FASTQ simulator with accurate TAPS chemistry — 9 samples, 4 biotypes, seeded m5C positions for end-to-end pipeline validation.

Experimental Design

Three-condition framework

sRNA-TAPS requires three sample conditions to correctly separate genuine m5C signal from chemistry background and sequencing noise.

treat

TET + Pyridine Borane

Full TAPS chemistry. Both 5mC and 5hmC are oxidised by TET and converted to T. This is your signal condition.

pb_Ctrl

Pyridine Borane only

No TET oxidation. Any C→T in these samples is chemistry background, not genuine modification. Used for δ score calculation.

no-treat

No chemistry

Untreated baseline. C→T at this level reflects sequencing error and SNPs, used to build the cell-line SNP blacklist.

The delta (δ) score = treat_mod_rate − pb_Ctrl_mod_rate. Subtracting the pb_Ctrl background isolates only TET-dependent signal, removing both global chemistry noise and sequence-context-specific biases at each position.

Installation

Up and running in minutes

Available on bioconda, PyPI, and from source. Conda is recommended for automatic dependency resolution.

🐍 Conda (recommended)

Installs all dependencies automatically

📦 pip

Python package installer

💻 From source

Clone from GitHub for development

terminal
# Install via conda (recommended)
conda install -c bioconda -c conda-forge srna-taps

# Or via pip
pip install sRNA-TAPS

# Install R packages for reports
Rscript install_R_packages.R

# Verify installation
srnataps --version
